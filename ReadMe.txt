*****Dependencies and requirement:
=================================

1. Netbeans 8.2.
2. Glassfish /Tomcat server 4.1.1 (for servlet).
3. WAMP/XAMP server (for PHP).
4. Apache’s commens API (for file upload).
5. CKEditor plug-in.
6. Jsoup library 1.9.2.
7. MySQL connector.
8. Oracle thin driver.
9. PostgreSQL library.
10.JDK 1.8



*****Prerequisite for deployment: 
================================

1. NetBeans IDE 8.2.
2. WAMP/XAMP server.
3. Glassfish/Tomcat server.

The system uses servlet and PHP files thus for executing servlet file we need Glassfish/Tomcat server and 
WAMP/XAMP server to execute PHP files.

*****Steps for deployment:
==========================
1. Paste the repository inside "WWW" directory (for WAMP) or "htdocs" directory (for XAMP).
2. Open the project in NetBeans IDE.
3. Start Glassfish/Tomcat server.
4. Start WAMP/XAMP server.
5. Enter homepage URL in browser like "./home.php" where "." is current path.
